AI/ML engineer with 2+ years shipping production systems across the full intelligence stack — transformer fine-tuning, LLM agent pipelines, RAG systems, and the backend and full-stack infrastructure that gets them in front of real users. Recognized with the **FedEx Innovation Award** for an NLP classifier that hit **94% accuracy**, cut case resolution by **30%**, and eliminated **40%** of manual triage across a two-team production pilot. My work spans every layer from model architecture and evaluation to microservice APIs and real-time dashboards — because the hardest problems in production AI aren't the models, they're the systems around them. Currently running as **Data Scientist II, AI Platform at FedEx Express**, with an M.S. in Computer Science from Georgia State University.
